# **App Name**: Gemini Captionator

## Core Features:

- Image Upload: Allow users to upload an image from their local device.
- Caption Generation: Send the uploaded image to the Gemini API and display the generated caption.
- Image and Caption Display: Display the uploaded image alongside the generated caption.

## Style Guidelines:

- Primary color: Neutral grays for a clean interface.
- Secondary color: Soft blues for accents and interactive elements.
- Accent color: Teal (#008080) to highlight important elements.
- Clean and modern layout with a focus on image presentation.
- Simple and clear icons for upload and other actions.
- Subtle animations for loading and transitions.
